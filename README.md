# Nail Art Booking & Catalog App

Esta es una aplicación web diseñada para la gestión de un negocio de manicura y diseño de uñas. Permite a las clientas visualizar un catálogo de diseños, consultar servicios y gestionar citas de manera eficiente.

## 💅 Características Principales

- **Catálogo Interactivo:** Galería de diseños de uñas categorizados (Nail Art, Gel, Acrílico, etc.).
- **Gestión de Citas:** Sistema para que las clientas puedan agendar y consultar disponibilidad.
- **Panel de Administración:** Control total sobre los servicios ofrecidos, precios y agenda.
- **Diseño Responsive:** Optimizado para dispositivos móviles y escritorio.

## 🚀 Tecnologías Utilizadas

- **Frontend:** React y CSS

## 🛠️ Instalación y Configuración

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/tu-usuario/paginauñas.git
   ```

2. **Instalar dependencias:**
   ```bash
   npm install
   ```

3. **Configurar variables de entorno:**
   Crea un archivo `.env` basado en `.env.example` y añade tus credenciales.

4. **Ejecutar la aplicación:**
   ```bash
   npm start
   ```
